# Shinobi Runner Game Website

A web-based 2D endless runner game where players control a shinobi to avoid obstacles, earn points, and compete for high scores. The game includes user registration, login, a dynamic scoreboard, and a fully interactive web interface.

## Features

- **User Authentication:** Register and log in to save high scores.
- **Interactive Gameplay:** Move left/right, jump, and avoid obstacles.
- **Scoring System:** Earn 15 points every 2 seconds; high scores are saved per user.
- **Dynamic Game Environment:** Randomized obstacle generation, smooth physics, and animations.
- **Scoreboard:** Displays top player scores; highlights the current user.
- **Audio Integration:** Jump and game over sounds enhance the experience.
- **Responsive Navigation:** Homepage, registration, login, game, and scoreboard pages.

## Game Instructions

- Move Right: Press → (right arrow)
- Move Left: Press ← (left arrow)
- Jump: Press ↑ (up arrow)
- Aim: Avoid spikes and survive as long as possible to achieve a high score.
- Restart: Press **R** after Game Over to restart the game.

## Technologies Used

- **HTML, CSS, JavaScript**
